# PD_Artjoms_Levcenko
Nosaukums: Ievarijuma izmaksas un svara kalkulators
Apraksts:
Programma saņem lietotaja ievaditus datus, abolu masu kg un cukura cena par 1 kg.
Programma apreķina cik kg cukura vajadzēs ievarījuma gatavošanai
Programma apreķina kopējo izmaksu EUR par cukuru
Programma apreķina ievarījuma masu ņemot vērā zūdumus.
Programma izvada rezultātus lietotājam.
Palaišanas instrukcija:
Programma uzrkastīta Python programmēšanas valodā. Jālaist failu Ivarijums_AL.py Visual Studio Code vidē.
Iespiež f5 klaviaturā izvelies Debugger. Ievadi datus, izmanto ciparus. Skaties rezultātu!
